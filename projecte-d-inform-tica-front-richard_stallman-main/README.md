## About The Project

> **PerfecT-shirt** es una tienda online especializada en el mundo del fútbol. Ofrecemos camisetas de todas las ligas, una sección de pujas exclusivas, cajas sorpresa (Random Box) y una herramienta para crear tu propia camiseta personalizada.
>
> Nuestro objetivo es brindar a los aficionados una experiencia de compra única con un diseño moderno, dinámico y fácil de usar.

## Getting Started

Para obtener una copia local y hacerla funcionar, sigue estos sencillos pasos.

### Prerequisites

Necesitarás un navegador web moderno y un servidor local si deseas probar las funcionalidades de carga dinámica de forma óptima.

- Navegador Web (Chrome, Firefox, Safari)
- Servidor local (opcional, p.ej. Live Server de VSCode)

### Installation

1. Clona el repositorio:

   ```sh
   git clone https://github.com/La-Salle-Gracia-FP/projecte-d-inform-tica-front-richard_stallman.git
   ```

2. Entra en la carpeta del proyecto:

   ```sh
   cd projecte-d-inform-tica-front-richard_stallman/perfecT-shirt
   ```

3. Ejecuta el archivo `index.html` en tu navegador o inicia un Live Server.

### Deployment

Si deseas desplegarlo en la web, puedes usar servicios como **GitHub Pages**, **Vercel** o **Netlify**, apuntando la raíz de publicación al directorio `perfecT-shirt`.

## Usage

PerfecT-shirt puede ser utilizado para múltiples funciones:

- **Catálogo:** Navegar y comprar camisetas de las principales ligas europeas e internacionales.
- **Pujas:** Participar en subastas de camisetas retro, históricas y ediciones limitadas.
- **Random Box:** Comprar cajas sorpresa con camisetas aleatorias según un presupuesto definido.
- **Personalización:** Diseñar una camiseta añadiendo tu nombre y dorsal preferido.
- **Ajustes de Preferencia:** Cambiar el tema visual a modo oscuro y seleccionar el idioma de la aplicación a tu gusto.

## Test

Al ser un proyecto frontend en HTML, CSS y JS, las pruebas se realizan de forma manual comprobando las interfaces y flujos de usuario directamente en el navegador. Las funcionalidades principales a probar incluyen:

- Añadir productos al carrito.
- Funcionamiento de la vista rápida.
- Activación del modo oscuro.
- Sistema de traducción integrado.

| Test Case          | Description                                                               |
| :----------------- | :------------------------------------------------------------------------ |
| Carrito de compras | Añadir y eliminar productos del carrito correctamente.                    |
| Vista rápida       | Comprobar el funcionamiento del modal de vista previa.                    |
| Modo oscuro        | Verificar la activación y persistencia del tema oscuro.                   |
| Traducción         | Comprobar el correcto funcionamiento del sistema de traducción integrado. |
